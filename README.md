## develop

`ndor` is impletioned with go, js and wasm.

If you want to run `ndor` on your local machineï¼š

1. Install `git` and `Go`.
2. Clone the repo

    ```shell
    git clone https://github.com/zrcoder/ndor
    ```

3. Run

    ```shell
    cd ndor
    go run ./cmd
    ```
